World Class Equine Applications

August 30, 2023

I developed a series of GUI applications to automate sections of the business workflow for World Class Equine. These applications integrate with the existing business Box Drive to save and retrieve data directly from PDFs already in use. The applications were written in Python using the PyQt framework and use the PyPDF2, pyebroidery, re, reportlab, typing, pathlib, svglib, json, and threading libraries.

The design goals of this project were entirely focused on functionality. While the applications are graphical, I invested no time into making them visually interesting. I wrote the programs to automatically complete any task they had the data for and only ask for user input for data that could not be retrieved. I developed the applications on my Linux system, then built the executables in a Windows virtual machine using pyinstaller.

Applications include:

  • A window screen overlay program that automatically overlays window screen graphics on PDF work-orders that require them.
  • An invoice processor that cleans up and compiles invoices from many locations into a simple summary. This process only takes a few seconds, while the manual process it replaced took several hours.
  • An embroidery lettering program that arranges and merges pre-made lettering embroidery files into final machine files. This allows the user to type the desired letters and the app will output the finished lettering embroidery file with consistent letter spacing.
  • A UPS batch file generator that reads a factory shipping request, finds relevant package data from sales order PDF records, and outputs a UPS batch file CSV that can be uploaded to the UPS site to create all the shipping labels at once.
