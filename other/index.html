Data Science Final Project

For the final project for CS 670 Data Science my teammate and I decided to analyze the distributions and characteristics of exoplanets in the NASA Exoplanet Archive dataset. I worked mostly on sections 2.4-2.8. My contributions primarily concerned the 3D plotting of the exoplanets. The 3D coordinates of the exoplanet observations were not part of the original dataset, so I used the Sesame Name Resolver to lookup the exoplanets coordinates from the name of their host star. I was then able to make a user-interactive 3D plot of the data.

World Class Equine Applications

I developed a series of GUI applications to automate sections of the business workflow for World Class Equine. These applications integrate with the existing business Box Drive to save and retrieve data directly from PDFs already in use. The applications were written in Python using the PyQt framework and use the PyPDF2, pyebroidery, re, reportlab, typing, pathlib, svglib, json, and threading libraries.

The design goals of this project were entirely focused on functionality. While the applications are graphical, I invested no time into making them visually interesting. I wrote the programs to automatically complete any task they had the data for and only ask for user input for data that could not be retrieved. I developed the applications on my Linux system, then built the executables in a Windows virtual machine using pyinstaller.

World Class Equine Website

World Class Equine Website

I redesigned and updated worldclassequine.com using the Hugo static site generator with the goal of minimizing the file size of the site and maximizing functionality and features with the added constraint of using zero JavaScript in the site. I decided to avoid JavaScript since it is a client side language and a site made without it contains inherently less security vulnerabilities and will typically run faster. I also believe that web development has shifted to overuse JavaScript and many sites use it unnecessarily.

NCUR Poster

I was invited to present a poster at NCUR (National Council on Undergraduate Research) 2020, but the event was canceled due to COVID. My research poster discusses the possibility of using tetrapod nano crystals as a strain-tunable single photon source and the process I went through to try and measure that using a Hanbury Brown & Twiss Interferometer. I made the graphics myself using a combination of Blender and Inkscape.
