CS 510 Game Programming Project

Proof-of-concept build:

The playable proof-of-concept of this project is extremely buggy, and the progression systems have not yet been implemented. I worked mostly on the ship asset, the inventory system, and the menus.


CS510 POC Build

Alpha build:

I almost completely refactored the inventory system for the alpha build. The game is much closer to having a single area implemented, but the main progression systems are not yet functional. I worked on the inventory and upgrade system, as well as all the assets.


CS510 Alpha Build

Beta build:

For this build I reworked the player menus, built a main menu, worked on a system for transitioning between zones, and almost finished the progression system. We decided to change the scope of the game from a trading game to a delivery game, since an economy would have added unnecessary complexity. However, we discovered too late that the new shader for the orbit lines was not compatible with webgl, so this version ended up even less playable then the alpha, despite being a lot more stable.


CS510 Beta Build

Final build:

In this build the orbit lines were reverted to their alpha state, and the game finally has a full feature set. Sound was added and several quality of life features were also implemented.
