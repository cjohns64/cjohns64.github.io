July 9, 2025

Mall Crawlers Project Summery

This game was made for the Godot Wild Jam #83. This was the first game jam I participated in. Our team of three people severely underestimated the time required to implement the vision we had for this project, so it ended up as more of a tech demo then a game. I spent most of my time working on making the assets, the level design, and the animation blend trees. I also did the light baking, this had a huge effect on the atmosphere of the game, but is also responsible for most of it’s file size.

Project Take-Aways

  • Managing scope is one of the most critical aspects of game design, and also very difficult to get right
  • Don’t start a project by only doing art assets, get the gameplay done first.
  • Remember to loop the background music before building the project
