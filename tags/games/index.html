Games

One-Armed Bandit!

2025

Top-down 2D game made for the Brackey's Game Jam 2025.2, the third game jam I participated in. This game got rated in the top 8% of over 2000 entries. You play as a clever one-armed raccoon sneaking around a neighborhood looking for biscuits in the dumpsters while avoiding the animal control officers. Each dumpster functions as a slot machine where each 'pull' is a dig through the trash.
