Tree Asset Project Summary

For this project I wanted to make a convincing CG tree that I could use in other projects. I had a lot of problems managing the complexity of the tree, since it is too computationally expensive and too time consuming to model each leaf. I ended up modeling a single branch, and then instancing that branch several times to form the tree. The instances were placed by hand.

I also did not use any transparency, since I had problems with transmission depth in past projects (Solaren Video Remake ground station scene).
