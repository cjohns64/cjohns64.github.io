Frigate Project Summary

I have always been fascinated by fully-rigged ships and for this project I made one in Blender. I learned about the rigging and layout of fully-rigged ships by reading Wikipedia articles. I didn’t notice the hull depth I was modeling was only to the water line of the reference ship until I was done with the rigging. I hadn’t setup the project organization correctly in order to support the large scale modifications needed to fix it. I learned a lot about how to use the non-destructive modeling workflow with this project.

I also encountered difficulties with making the ocean. The two main issues were getting it to convincingly end at the horizon without being to unmanageable and making the ocean look like a large body of water.
